# 補助金業務 運用マニュアル

## はじめに

このマニュアルでは、IT導入補助金の案件管理から申請書類作成までの一連の流れを説明します。
ほとんどの作業は自動化されているので、手動で行うのは「管理表でのステータス更新」と「書類の最終確認」が中心です。

---

## 全体の流れ

```
1. 支援事業者が送客フォームを送信
      ↓（自動）
2. 案件管理表にデータが転記される
   Google Driveに顧客フォルダが作成される
      ↓（手動）
3. 案件管理表でステータスを更新しながら進捗管理
      ↓（自動）
4. メール添付ファイルがDriveに自動格納される
      ↓（手動）
5. 書類自動作成ツールで申請書類を作成
      ↓（手動）
6. 出力された書類を目視確認して申請
```

---

## STEP 1: 送客フォームの送信

支援事業者（ピスケスなど）が新規顧客の情報をフォームから送信します。

### フォーム送信後に自動で行われること

| 処理 | 内容 |
|------|------|
| 管理表への転記 | お客様企業名、申請枠、支援事業者名、連絡先がシート1に自動入力される |
| プルダウン設定 | CB担当、ステータス、公募回などのプルダウンが自動で設定される |
| チェックボックス設定 | 書類確認用のチェックボックスが自動で設定される |
| Driveフォルダ作成 | 支援事業者フォルダの下に顧客フォルダが自動作成される |
| リンク設定 | 管理表のB列（お客様企業名）をクリックするとDriveフォルダに飛べる |

### 注意点
- フォーム送信後、管理表への反映は数秒〜十数秒かかります
- 同じ企業名で再送信した場合、Driveフォルダは重複作成されません

---

## STEP 2: 案件管理表での進捗管理

### アクセス
https://docs.google.com/spreadsheets/d/1YKHps9kq7gQ9kZIXXyiukfq_qMHN56NxP1J_f-9hQpU/edit

### シート1の主な列

| 列 | 内容 | 操作方法 |
|---|------|---------|
| A: CB担当 | カラフルボックス側の担当者 | プルダウンから選択 |
| B: お客様企業名 | 顧客名（クリックでDriveフォルダへ） | 自動入力される |
| C: 構成員/支援事業者名 | 支援事業者名 | 自動入力される |
| D: 申請枠 | 通常枠/インボイス枠 | 自動入力される |
| E: 公募回 | 何回目の公募か | プルダウンから選択 |
| F: ステータス | 現在の進捗 | プルダウンから選択 |
| G: 進捗・状況詳細 | 補足メモ | 自由に入力 |

### ステータスの流れ

案件の進行に合わせて、F列のステータスを更新していきます。

```
初回電話案内（未）
  → 申請_資料待ち
  → 申請_CB準備中
  → 申請_準備完了        ← ここでGoogle Chatに通知が届く
  → 交付申請 申請済
  → 交付決定（または不採択）
  → 実績報告_案内済
  → ...
  → 補助金 交付済
```

### Google Chat通知について
- ステータスを「申請_準備完了」に変更すると、Chatスペース「補助金連絡」に自動通知が届きます
- クラフトバンクの案件は通知対象外です

### ダッシュボード
案件管理表の「ダッシュボード」タブで、ステータス別・支援事業者別・申請枠別などの件数が確認できます。

## STEP 3: Driveのフォルダ構造

フォーム送信時に、以下の構造でフォルダが自動作成されます。

```
01_補助事業者サポート/
  2026/
    株式会社ピスケス/              ← 支援事業者ごとのフォルダ
      001.〇〇株式会社_通常枠/     ← 顧客フォルダ（連番付き）
        1.交付申請/
        2.実績報告/
      002.△△株式会社_インボイス枠/
        1.交付申請/
        2.実績報告/
    いろはな株式会社/
      001.□□株式会社_通常枠/
        ...
```

- 管理表のB列（お客様企業名）をクリックすると、該当の顧客フォルダに直接アクセスできます
- 書類はこのフォルダに格納していきます

---

## STEP 4: メール添付ファイルの自動格納

補助金関連のメールに添付されたファイルが、15分おきにDriveの該当フォルダに自動格納されます。

### 条件
- Gmailのラベル「補助金_受信」がついたメールが対象
- 添付ファイルがあるもの

### 注意点
- 15分間隔なので、すぐに反映されない場合は少しお待ちください
- どの顧客フォルダに入るかは、メールの件名や送信元から自動判定されます

## STEP 5: 書類自動作成ツール（Streamlit）

### アクセス
https://hojokin-uymden838zfglt9uahkapf.streamlit.app

しばらくアクセスがないとスリープしています。「Yes, get this app back up!」ボタンを押して1分ほどお待ちください。

### 使い方

1. 会社名とテンプレート（通常枠 or インボイス枠）を選択
2. データの読み込み方法を選ぶ

| 方法 | いつ使う |
|------|---------|
| ファイルアップロード | 手元にファイルがある場合 |
| Google Drive | 顧客フォルダから直接読み込む場合（フォルダを選ぶだけ） |

3. ヒアリングシート（PDF）や賃金台帳（Excel）などを読み込む
4. AIが自動で内容を読み取り、申請テンプレートに転記
5. 出力されたExcelファイルをダウンロード

### 対応しているファイル
- ヒアリングシート（PDF）
- 賃金台帳（Excel） — 3種類のフォーマットに対応
- 履歴事項全部証明書（PDF）
- 納税証明書（PDF）
- 損益計算書（PDF）

### 重要: 必ず目視確認すること
- AIが読み取った内容には誤りが含まれる可能性があります
- 特に**金額**と**日付**は必ずチェックしてください
- 読めないフォーマットがあれば共有してください

## よくあるトラブルと対処法

| 症状 | 対処 |
|------|------|
| フォーム送信後、管理表に反映されない | 数十秒待ってページをリロード |
| Driveフォルダが作られない | Apps Scriptの実行ログでエラーを確認 |
| Streamlitアプリが開けない | スリープ中。ボタンを押して1分待つ |
| 賃金台帳が読めない | 新しいフォーマットの可能性。サンプルを共有ください |
| Chat通知が届かない | ステータスが「申請_準備完了」になっているか確認 |
| B列クリックでDriveに飛べない | リンクが設定されていない可能性。羽根に連絡ください |
