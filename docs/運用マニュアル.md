# 補助金業務 運用マニュアル

## 全体フロー

```
送客フォーム送信
    ↓
┌───────────────────────────────────┐
│ 自動処理（GAS）                     │
│                                     │
│  1. 案件管理表「シート1」にデータ転記  │
│     - プルダウン・チェックボックス自動設定 │
│  2. Google Driveにフォルダ自動作成     │
│     - 支援事業者名/顧客名_申請枠/     │
│       - 1.交付申請/                  │
│       - 2.実績報告/                  │
│  3. 管理表の企業名にフォルダリンク設定  │
└───────────────────────────────────┘
    ↓
案件管理表で進捗管理（手動）
    ↓
補助金書類自動作成ツール（Streamlit）で申請書類作成
```

---

## 1. 送客フォーム → 管理表・Drive

### 何が起きるか
支援事業者がフォームを送信すると、以下が**自動で**行われます。

| 処理 | 内容 |
|------|------|
| 管理表への転記 | お客様企業名、申請枠、支援事業者名、連絡先などがシート1に自動入力されます |
| プルダウン設定 | CB担当、ステータス、公募回などのプルダウンが自動で設定されます |
| チェックボックス設定 | 書類確認用のチェックボックスが自動で設定されます |
| Driveフォルダ作成 | 支援事業者名のフォルダの下に、顧客フォルダが自動作成されます |
| リンク設定 | 管理表のB列（お客様企業名）をクリックすると、Driveフォルダに飛べます |

### Driveのフォルダ構造
```
01_補助事業者サポート/
  2026/
    株式会社ピスケス/
      001.〇〇株式会社_通常枠/
        1.交付申請/
        2.実績報告/
      002.△△株式会社_インボイス枠/
        ...
    いろはな株式会社/
      001.□□株式会社_通常枠/
        ...
```

### 注意事項
- フォーム送信後、管理表への反映は数秒～十数秒かかります
- 同じ企業名で再送信しても、Driveフォルダは重複作成されません
- 支援事業者名のフォルダがまだない場合は自動で作成されます

## 2. メール添付ファイルの自動格納

### 何が起きるか
補助金関連のメールに添付されたファイルが、**15分おき**に自動でDriveの該当フォルダに格納されます。

### 条件
- Gmailのラベル「補助金_受信」がついたメール
- 添付ファイルがあるもの

### 注意事項
- 自動格納は15分間隔です。すぐに反映されない場合は少しお待ちください
- どの顧客フォルダに入るかは、メールの件名や送信元から自動判定されます

## 3. 案件管理表の使い方

### アクセス
案件管理表（2026年度案件管理表）のシート1で進捗管理を行います。

### 主な列
| 列 | 内容 | 操作 |
|---|------|------|
| A: CB担当 | 担当者（プルダウン） | 手動で選択 |
| B: お客様企業名 | 顧客名（Driveリンク付き） | 自動入力 |
| C: 構成員/支援事業者名 | 支援事業者（プルダウン） | 自動入力 |
| D: 申請枠 | 通常枠/インボイス枠 | 自動入力 |
| E: 公募回 | 申請回（プルダウン） | 手動で選択 |
| F: ステータス | 進捗状況（プルダウン） | 手動で更新 |
| G: 進捗・状況詳細 | 備考 | 手動で入力 |

### ステータスの流れ
```
初回電話案内（未）
  → 申請_資料待ち
  → 申請_CB準備中
  → 申請_準備完了
  → 交付申請 申請済
  → 交付決定（または不採択）
  → 実績報告_案内済
  → ...
  → 補助金 交付済
```

## 4. 補助金書類自動作成ツール（Streamlit）

### アクセス
https://hojokin-uymden838zfglt9uahkapf.streamlit.app

### 使い方
1. 「ファイルアップロード」または「Google Drive」を選択
2. ヒアリングシート（PDF）や賃金台帳（Excel）を読み込む
3. AIが自動で申請テンプレートに転記
4. 出力されたExcelファイルをダウンロード

### データソースの選び方
| 方法 | 用途 |
|------|------|
| ファイルアップロード | 手元にファイルがある場合 |
| Google Drive | 共有ドライブから直接読み込む場合（顧客フォルダを選ぶだけ） |

### 注意事項
- AIが読み取った内容は**必ず目視で確認**してください（特に金額・日付）
- 賃金台帳は3種類のフォーマットに対応していますが、読めない形式があれば共有ください
- しばらくアクセスがないとアプリがスリープします。「Yes, get this app back up!」ボタンを押して1分ほど待ってください

## トラブルシューティング

| 症状 | 原因・対処 |
|------|----------|
| フォーム送信後、管理表に反映されない | 数十秒待ってリロード。それでもダメな場合はApps Scriptの実行数を確認 |
| Driveフォルダが作られない | Apps Scriptの実行数でエラーを確認 |
| Streamlitアプリが開けない | スリープ中。ボタンを押して待つ |
| 賃金台帳が読めない | 新しいフォーマットの可能性。サンプルを共有ください |
| プルダウンが表示されない | 通常は自動設定。表示されない場合は連絡ください |

## 管理者向け情報

### Apps Script
案件管理表 → 拡張機能 → Apps Script で開けます。

| ファイル | 役割 |
|---------|------|
| コード.gs | フォーム回答 → シート1への転記 |
| folder_creator.gs | Driveフォルダ自動作成 + リンク設定 |
| mail_to_drive.gs | メール添付ファイルの自動格納 |

### トリガー
| 関数 | トリガー種類 | タイミング |
|------|------------|-----------|
| onFormSubmit | フォーム送信時 | フォーム送信のたび |
| onFormSubmitFolder | フォーム送信時 | フォーム送信のたび |
| processNewEmails | 時間主導型 | 15分おき |
