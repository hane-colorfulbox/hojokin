# GAS（Google Apps Script）自動化

## 概要
補助金業務の資料管理を自動化するGoogle Apps Script。

## 機能
1. **メール添付ファイル自動格納** (`mail_to_drive.gs`)
   - 資料提出メールの添付ファイルをGoogle Driveに自動保存
   - 案件管理表にチェックを入れる

2. **送客フォーム自動処理** (`form_to_sheet.gs`)
   - Googleフォームの回答を管理シートに自動転記
   - 顧客用Driveフォルダを自動作成

## セットアップ
1. Google Apps Script (https://script.google.com) で新規プロジェクト作成
2. 各 .gs ファイルの内容をコピー
3. CONFIG の値を実際のID/URLに変更
4. トリガーを設定（手順は各ファイル内のコメント参照）
