# 補助金申請書類 作成マニュアル

## このマニュアルについて

Claude Code（VSCode拡張機能）を使って、IT導入補助金の申請書類を半自動で作成する手順書です。
Claude Codeに詳しくなくても、このマニュアルの通りにやれば大丈夫です。

---

## 1. 事前準備（初回のみ）

### 1-1. VSCodeでClaude Codeを使えるようにする

1. VSCodeを開く
2. 左のアイコンバーから拡張機能（四角いマーク）をクリック
3. 検索欄に「Claude Code」と入力
4. 「Claude Code」をインストール
5. VSCode左下にClaudeのアイコンが出たらOK

### 1-2. フォルダ構成

作業フォルダは以下の場所に統一してください:

```
c:/Users/[自分のユーザー名]/projects/カラフルボックス/補助金/
├── 【原本_法人】企業名_通常枠_法人2026.xlsx      ← テンプレート（触らない）
├── 【原本_法人】企業名_インボイス枠_法人2026.xlsx  ← テンプレート（触らない）
│
├── [顧客名]資料/                  ← 顧客ごとにフォルダを作る
│   ├── ヒアリングシート_xxx.xlsx
│   ├── 履歴事項全部証明書_xxx.pdf
│   ├── 納税証明書_xxx.pdf
│   ├── 損益計算書_第1期_xxx.pdf
│   ├── 損益計算書_第2期_xxx.pdf
│   ├── 見積書_xxx.xlsx
│   ├── 給与データ_xxx/            ← 月次の支給控除一覧PDF
│   │   ├── 3月度支給控除一覧.pdf
│   │   ├── 4月度支給控除一覧.pdf
│   │   └── ...
│   └── 提出資料/
│       ├── 賃金台帳.pdf
│       └── 賃金状況報告シート.xlsx
```

**ファイル名のルール:**
- ファイル名にはなるべく顧客名を入れる（Claudeがどの会社の資料か判別できるように）
- ファイル名は日本語でOK

---

## 2. 申請書作成（タスク1）

ヒアリングシート・登記簿・決算書などから、申請書テンプレートに自動転記します。

### 手順

1. **VSCodeを開く**
2. **Claude Codeのチャット欄を開く**（左下のClaudeアイコンをクリック）
3. **下のプロンプトをコピーして、★の部分だけ書き換えて送信**

---

### コピペ用プロンプト（申請書作成）

```
補助金の申請書類を作成してください。

■ 顧客名: ★顧客名をここに入れる★
■ 申請枠: ★「通常枠」か「インボイス枠」を入れる★
■ 資料フォルダ: c:/Users/★自分のユーザー名★/projects/カラフルボックス/補助金/★顧客名★資料/
■ テンプレート: c:/Users/★自分のユーザー名★/projects/カラフルボックス/補助金/【原本_法人】企業名_★枠名★_法人2026.xlsx

■ 作業内容:
1. 資料フォルダの中身を確認してファイル一覧を出す
2. テンプレートExcelのコピーを作成（原本は絶対に変更しない）
3. 以下の資料を読み取ってテンプレートに転記:
   - ヒアリングシート → 申請内容シートの各項目
   - 履歴事項全部証明書(PDF) → 法人名・所在地・代表者・設立日・資本金
   - 納税証明書(PDF) → 納税額
   - 損益計算書(PDF) → 売上高・営業利益・経常利益・給料手当・雑給・賞与・減価償却費 → 生産性指標計算シートにも転記
   - 見積書(xlsx) → 申請金額
4. AI判断が必要な項目も可能な限り埋める:
   - 業種コード（事業内容から推定）
   - 経営状況の判定（営業利益がマイナスなら「事業維持に注力」）
   - 地域別最低賃金（所在地から判定）
5. 出力ファイル名: ★顧客名★_★枠名★_AI版.xlsx（資料フォルダ内に保存）
6. 完了後、空欄のセルが何件あるか報告する

■ 注意:
- テンプレート原本のデータは使わない。必ず資料から読み取った値だけを使う
- Pythonのopenpyxlで処理する
- 日本語パスはpathlib.Pathのiterdir()で解決する
- sys.stdout.reconfigure(encoding='utf-8')をスクリプト先頭に入れる
- Excelに書き込む文字列の先頭に「=」を入れない（Excelが数式と誤認するため）
```

---

4. **Claudeが作業を始めるので待つ**（10-20分）
   - 途中で「ツールの実行を許可しますか？」と聞かれたら「許可」を押す
   - 何度か聞かれるので、その都度「許可」でOK
5. **完了したら出力Excelを開いて確認**
6. **空欄が残っている箇所をメモ** → ミーティングで確認する項目

### 空欄が残りやすい項目（正常です）

- 役員のフリガナ（資料に読み仮名がないため）
- アンケート回答欄
- ミーティングでの確認事項（支援事業者との相談項目）
- ファイル添付・操作系の項目（実際の申請画面で操作するもの）

## 3. 給与支給総額の計算（タスク2）

月次の給与データと損益計算書から、給与支給総額と1人当たり給与支給総額を計算します。

### 手順

1. **Claude Codeのチャット欄を開く**
2. **下のプロンプトをコピーして、★の部分だけ書き換えて送信**

---

### コピペ用プロンプト（給与支給総額計算）

```
給与支給総額の計算Excelを作成してください。

■ 顧客名: ★顧客名をここに入れる★
■ 資料フォルダ: c:/Users/★自分のユーザー名★/projects/カラフルボックス/補助金/★顧客名★資料/

■ フォルダ内にある資料:
- 月次の給与支給控除一覧（PDF）: 直近3ヶ月以上
- 損益計算書（PDF）: 直近の決算期
- 賃金状況報告シート（xlsx）: あれば使う

■ 作業内容:
1. 資料フォルダの中身を確認してファイル一覧を出す
2. 損益計算書PDFから以下を読取:
   - 給料手当、雑給、賞与（販管費の内訳）
   - 売上高、営業利益、経常利益、減価償却費
3. 給与支給控除一覧PDFから以下を読取:
   - 従業員ごとの氏名、雇用形態（正社員/パート）、課税支給合計
4. 賃金状況報告シートがあればそこから従業員データを取得
5. 以下を計算:
   - 給与支給総額 = 給料手当 + 雑給 + 賞与（P/Lから）
   - 役員報酬を特定して控除した金額も算出
   - パート・アルバイトのFTE換算（実労働時間 ÷ 標準2080時間/年）
   - 1人当たり給与支給総額 = 給与支給総額 ÷ FTE換算後従業員数
6. 賃上げ計画シミュレーション（年率1.5%増の3年計画）
7. 2026テンプレートに転記する値の一覧も出す

■ 出力ファイル: ★顧客名★_給与支給総額計算.xlsx（資料フォルダ内に保存）
■ シート構成: 給与支給総額計算 / 従業員別明細 / 賃上げ計画

■ 注意:
- Pythonのopenpyxlで処理する
- 日本語パスはpathlib.Pathのiterdir()で解決する
- sys.stdout.reconfigure(encoding='utf-8')をスクリプト先頭に入れる
- Excelに書き込む文字列の先頭に「=」を入れない
- 給与支給総額から除外する項目: 福利厚生費、法定福利費、退職金
```

---

3. **Claudeが作業を始めるので待つ**（10-15分）
4. **出力Excelを開いて確認** → 特に以下をチェック:
   - 従業員数が合っているか
   - 損益計算書の金額が正しく読み取れているか
   - 1人当たり給与支給総額の値

## 4. うまくいかないとき

### よくあるトラブルと対処法

**Q: 「許可しますか？」が何度も出てうっとうしい**
A: 「常に許可」を選べば、以降は自動で実行されます。

**Q: エラーが出て止まった**
A: Claudeに「エラーが出たので修正して」と送ればOK。大抵は自動で直してくれます。

**Q: Excelを開いたら「内容に問題が見つかりました」と出る**
A: 「はい」を押せば開けます。問題なければそのまま使えます。
もし壊れていたら、Claudeに「Excelファイルが壊れているので修正して再出力して」と送ってください。

**Q: 数値が間違っている気がする**
A: Claudeに「○○の値が違うと思う。損益計算書をもう一度確認して」と具体的に伝えてください。

**Q: 途中で止まって何も起きない**
A: チャット欄に「続けて」と送ってください。

**Q: 全然違うことをやり始めた**
A: チャット欄に「止めて。最初からやり直して」と送ってから、もう一度プロンプトを貼り直してください。

## 5. 出力ファイルの使い方

### 申請書（_AI版.xlsx）
- そのまま申請に使う前に必ず担当者がチェック
- 空欄はミーティングで確認して手入力
- 生産性指標計算シートの計画数値は必要に応じて調整

### 給与支給総額計算（_給与支給総額計算.xlsx）
- 「2026テンプレート転記用」セクションの値を申請書テンプレートに転記
- 1人当たり給与支給総額は「推奨」の行を使う
- 賃上げ計画シートの年率1.5%は要件最低ライン。必要に応じて調整
